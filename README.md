# Vault - Balancer V2 --- Hardhat Framework

## Requeriments

Please, check that node v16.13.0 is installed (hardhat doesnt work with v17 right now) and hardhat is installed.


## Compile

```
npx hardhat compile
```


## Test

No test files yet
